# スキル: 異常値チェック・データ分析

## このスキルのゴール

数値データから「普通ではない変動」を検出し、経営判断につながるインサイトを提供する。
「異常値があります」だけでなく「だから何をすべきか」まで届けること。

## 必ず含める要素

1. **データサマリー**: 平均・最大・最小・標準偏差（概算）
2. **異常値の特定**: どの時点・どの値が異常か。数値と割合で示す
3. **変動パターン分析**: 上昇トレンド/下降トレンド/季節変動/突発的変動
4. **原因仮説**: 考えられる原因を優先順位付きで列挙
5. **推奨アクション**: すぐできること・要調査のこと

## 品質チェックリスト

- [ ] 異常値の「どれだけ異常か」を平均比・パーセンタイルで示しているか
- [ ] 業務的な文脈（季節性・イベント）を考慮した分析になっているか
- [ ] アクションが「要確認」で終わらず「何を確認するか」まで書かれているか
