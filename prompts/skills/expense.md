# スキル: 経費精算サポート

## このスキルのゴール

経費精算の申請内容を整理し、適切な勘定科目・申請方法を案内する。
申請者が迷わず、経理担当者が確認しやすい形で整える。

## 必ず含める要素

1. **経費の種別判断**: 何費に該当するか・会社負担の対象か
2. **勘定科目**: 正しい科目の選択と根拠
3. **消費税区分**: 課税/非課税/不課税
4. **申請に必要な書類**: 領収書・申請書・添付資料
5. **申請の注意点**: よくある却下理由・確認事項

## 主要な経費科目と判断基準

| 支出 | 勘定科目 | 課税区分 | 注意点 |
|------|---------|---------|--------|
| 飲食（得意先） | 接待交際費 | 課税10% | 参加者・目的を記載 |
| 飲食（社内打合せ） | 会議費 | 課税10% | 参加者・議題を記載 |
| 電車・バス | 旅費交通費 | 課税10% | ICカード明細でOK |
| タクシー | 旅費交通費 | 課税10% | 乗車理由を記載 |
| 出張宿泊 | 旅費交通費 | 課税10% | 出張報告書と紐付け |
| 書籍 | 新聞図書費 | 課税10% | 業務関連のもののみ |
| セミナー参加費 | 研修費 | 課税10% | 業務関連のもののみ |
| 慶弔費 | 福利厚生費/接待交際費 | 課税10%/非課税 | 相手によって異なる |
| 備品（10万円未満） | 消耗品費 | 課税10% | 1品10万円以上は固定資産 |
| 備品（10万円以上） | 固定資産 | 課税10% | 減価償却が必要 |

**経費内容**: 〇〇
**金額**: ¥X,XXX（税込）

---

### 判断結果

## 品質チェックリスク

- [ ] 個人的な支出が混入していないか（プライベートの飲食等）
- [ ] 一人での飲食が接待費になっていないか
- [ ] 10万円以上の備品が消耗品費になっていないか
