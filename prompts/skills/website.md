# スキル: Webサイト制作相談

## このスキルのゴール

Webサイトを作ったことがない人でも迷わず発注・制作を進められるよう、全体像を整理する。
「どんな方法が・いくらで・どれくらいで・どんな手順で」できるかを明確に伝えること。

## 必ず含める要素

1. **推奨制作方法**: 目的・予算・スキルに合った方法の選定と理由
2. **必要な機能・ページ**: 目的達成に必要な最小構成
3. **費用の目安**: 方法別の初期費用・月額費用のレンジ
4. **制作の流れ**: ステップバイステップのロードマップ
5. **よくある失敗・注意点**: 後悔しないためのポイント

## 制作方法の選択基準

| 方法 | 予算感 | 向いている場合 |
|------|--------|-------------|
| Wix/Squarespace | 月2,000〜5,000円 | 簡単なHP・情報発信のみ |
| WordPress（テンプレ） | 初期10〜30万円 | 中小企業の標準的なサイト |
| WordPress（カスタム） | 初期30〜100万円 | 独自デザイン・複雑な機能 |
| フルスクラッチ | 100万円〜 | EC・予約システム等の複雑な機能 |
| ノーコードツール | 無料〜月5,000円 | LP・ランディングページのみ |
