# スキル: チラシ・印刷物作成

## このスキルのゴール

捨てられずに読まれ、行動につながるチラシの構成・コピーを作る。
デザイナーへの発注書・指示書としてそのまま使える具体性を目指す。

## チラシの黄金構造（PASONA）

1. **Problem（キャッチコピー）**: 「この悩み、ありませんか？」で読者を止める
2. **Agitation（現状の問題）**: 放置した場合の損失・リスクを具体的に
3. **Solution（解決策）**: このサービスで何が解決するか
4. **Narrow（限定・特典）**: 今だけ・先着・地域限定等
5. **Action（CTA）**: 電話番号・URL・QRコードへの誘導

## 必ず含める要素

1. **キャッチコピー（大見出し）**: 1行で読者の心を掴む。問いかけ形式が有効
2. **サブコピー（小見出し）**: キャッチの補足・具体化
3. **本文**: サービス内容・特徴3点
4. **お客様の声・実績**: 信頼を作る
5. **CTA**: 具体的な行動の指示（電話・LINE・来店等）
6. **デザイン指示**: 色・写真・レイアウトの方向性

```

**サイズ**: A4 / A5 / B5（指定に従う）
**ターゲット**: 〇〇
**目的**: 〇〇（来店・電話・LINE追加 etc.）

〇〇のためのサービスです。〇〇の方に選ばれています。

**【本文：特徴3点】**
✅ 〇〇（理由・根拠）
✅ 〇〇（理由・根拠）
✅ 〇〇（理由・根拠）

**【実績・信頼】**

**【限定オファー】**

---

### デザイン指示

**カラー**: メイン〇〇色、アクセント〇〇色
**写真・イラスト**: 〇〇（スペース確保箇所を指示）

---

```

## 品質チェックリスト

- [ ] キャッチコピーが5秒以内に読めるか
- [ ] ターゲットが「自分のことだ」と感じるコピーか
- [ ] CTAが1つに絞られているか（電話とURLとLINEを全部並べない）
- [ ] 期限・限定感が入っているか
