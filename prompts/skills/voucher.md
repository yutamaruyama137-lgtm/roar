# スキル: 証憑読み取り・仕訳作成

## このスキルのゴール

領収書・請求書・納品書などの証憑を読み取り、すぐに会計ソフトに入力できる仕訳データを作成する。
入力の曖昧さを極力なくし、経理担当者の確認工数を最小化すること。

## 必ず含める要素

1. **仕訳（借方・貸方）**: 勘定科目・金額・税区分
2. **摘要**: 「何のために使ったか」が後で見てわかる内容
3. **消費税区分**: 課税/非課税/不課税/免税 + 税率（10%/8%軽減等）
4. **税込・税抜の内訳**: 本体価格と消費税額を分けて明示
5. **注意事項**: 判断に迷う点・追加確認が必要な事項

## 出力フォーマット

```
## 仕訳データ

**取引日**: YYYY年MM月DD日

| 区分 | 勘定科目 | 補助科目 | 金額 | 税区分 |
|------|---------|---------|------|--------|
| 借方 | 〇〇 | 〇〇 | ¥X,XXX | 課税10% |
| 貸方 | 〇〇 | 〇〇 | ¥X,XXX | - |

**摘要**: 〇〇に関する〇〇代

**税額内訳**
- 税抜金額: ¥X,XXX
- 消費税額: ¥XXX（10%）
- 税込合計: ¥X,XXX

---

## 勘定科目の選択根拠

〇〇を選んだ理由: （簡単な説明）

---

## ⚠️ 確認が必要な事項

- 〇〇が不明確なため、〇〇を確認してください
```

## よく使う勘定科目（参考）

| 支出の種類 | 勘定科目 | 税区分 |
|-----------|---------|--------|
| 飲食（得意先接待） | 接待交際費 | 課税10% |
| 飲食（社内打ち合わせ） | 会議費 | 課税10% |
| 電車・バス | 旅費交通費 | 課税10% |
| タクシー | 旅費交通費 | 課税10% |
| 書籍・雑誌 | 新聞図書費 | 課税10% |
| 事務用品 | 消耗品費 | 課税10% |
| ソフトウェアライセンス（月額） | 通信費 or 支払手数料 | 課税10% |
| 家賃 | 地代家賃 | 非課税 |
| 給与 | 給与賃金 | 不課税 |

## 品質チェックリスト

- [ ] 借方・貸方の合計金額が一致しているか
- [ ] 消費税額の計算が正しいか
- [ ] 摘要が「〇〇代」だけでなく目的がわかるか
- [ ] 判断が難しい場合は「要確認」として明示しているか
