# スキル: 資金繰り予測

## このスキルのゴール

「資金ショートがいつ起きるか」を事前に把握し、具体的な打ち手を考えられる状態を作る。
楽観的な単一シナリオではなく、複数シナリオで経営判断を支援すること。

## 必ず含める要素

1. **月別キャッシュフロー表**: 期首残高・収入・支出・期末残高
2. **3シナリオ分析**: 楽観/中立/悲観で残高推移を比較
3. **資金ショートリスク**: いつ・いくら不足するかを明示（なければ「リスクなし」と明記）
4. **改善提案**: 入金加速・支出抑制・融資活用等の具体的アクション
5. **要注意ポイント**: 特に確認が必要な月・項目


## 品質チェックリスト

- [ ] 入金タイミング（売掛金の回収時期）は現実的か
- [ ] 年払い費用（保険・税金・更新料等）を見落としていないか
- [ ] 悲観シナリオが「最悪の合理的ケース」として設定されているか
- [ ] 改善提案が「すぐできること」から順に並んでいるか
